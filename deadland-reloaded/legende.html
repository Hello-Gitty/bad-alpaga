Legende

Les Jetons blancs

    S'utilisent:
    • pour recuperer immediatement et a tout moment d'un etat Secoue ( meme apres un jet de Vigueur rate )
    • pour relancer un jet de Trait en gardant le meilleur resultat

Les Jetons rouges

    S'utilisent:
    • comme les Jetons blancs
    • pour lancer un d6 et d’ajouter son resultat a ton total ( il peut comme d’habitude faire un As )
    • pour annuler un echec critique

    Contrecoup : quand il est utilise par un Hombre le Marshal tir un jeton ( sauf s'il a ete utilise comme un jeton Blanc )

Les Jetons bleus

    S'utilisent:
    • comme les Jetons rouges mais sans le "Contrecoup"
    • pour piocher une carte aventure

Les Jetons legendaires

    Ils sont a usage unique (il ne revient pas dans le pot une fois utilise) et s'utilisent:
    • comme les Jetons bleu
    • s'il est utilise pour ameliorer ou refaire un jet il ajoute +2 au resultat
    • pour relancer n’importe quel jet ( un jet de degats, un jet sur une table )
    • en fin de la partie, pour obtenir une Prime
